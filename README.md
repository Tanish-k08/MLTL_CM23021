## üß™ List of Practicals / Experiments

### 1. TensorFlow.js Setup & Basics
- Verify TensorFlow.js installation  
- Create tensors, perform mathematical operations, and print results  

### 2. Linear Regression with Synthetic Data
- Train a simple regression model using synthetic data  
- Implement model using `tf.sequential()` and `tf.layers.dense()`

### 3. Digit Recognition (MNIST Dataset)
- Train a Convolutional Neural Network (CNN)  
- Evaluate predictions on handwritten digit images  

### 4. Text Sentiment Analysis
- Classify text as **positive** or **negative**  
- Use RNN or pre-trained sentiment analysis models  

### 5. Image Classification using Pre-trained Model (MobileNet)
- Load MobileNet model  
- Classify images directly in the browser  

### 6. Webcam-Based Object Detection
- Access webcam using Web APIs  
- Capture real-time frames and classify objects  

### 7. Pose Detection using PoseNet
- Detect human body keypoints  
- Perform real-time pose detection from webcam feed  

### 8. Model Deployment in Browser
- Save trained models locally  
- Reload models and run predictions in browser  

### 9. Transfer Learning with Images
- Retrain MobileNet for custom image categories  
- Example: fruit classification  

### 10. Open-Ended Mini Project
- Design and implement a real-world ML web application  
- Apply learned TensorFlow.js concepts
